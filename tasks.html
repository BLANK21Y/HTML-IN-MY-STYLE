HTML Learning Path

Master HTML5 with a structured learning approach

Complete HTML Learning Tasks

This comprehensive learning path will guide you from beginner to advanced HTML concepts through practical tasks. Follow the structured approach to build a solid foundation in HTML5.

Beginner

1–10 tasks | ~3–5 days
Task 1

Create a basic HTML page

Learn the fundamental structure of an HTML document including DOCTYPE, html, head, and body elements.

<!DOCTYPE html>
<html>
<head>
<title>My Page</title>
</head>
<body>
</body>
</html>
Task 2

Add headings and paragraphs

Learn to use heading tags (h1-h6) and paragraph tags to structure content.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>This is a paragraph.</p>
Task 3

Use formatting tags

Apply text formatting using bold, italic, underline, strong, and emphasis tags.

<b>Bold</b>
<i>Italic</i>
<u>Underline</u>
<strong>Strong</strong>
<em>Emphasis</em>
Task 4

Insert links

Create hyperlinks to connect pages and resources using anchor tags.

<a href="https://example.com">Visit Example</a>
Task 5

Add images

Insert and configure images in your HTML documents.

<img src="image.jpg" alt="Description" width="300" height="200">
Task 6

Create lists

Build ordered and unordered lists to organize content.

<ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>
Task 7

Create tables

Build tables to display structured data using table, tr, td, and th elements.

<table>
<tr>
<th>Header</th>
</tr>
<tr>
<td>Data</td>
</tr>
</table>
Task 8

Use comments

Add comments to your HTML code for documentation and organization.

<!-- This is a comment -->
Task 9

Add horizontal line and line breaks

Use horizontal rule and line break elements for layout control.

<hr>
Line 1<br>Line 2
Task 10

Add metadata

Configure document metadata using meta and title tags in the head section.

<head>
<title>Page Title</title>
<meta charset="UTF-8">
<meta name="description" content="Description">
</head>

Intermediate

11–20 tasks | ~5–7 days
Task 11

Create a contact form

Build interactive forms with various input elements and controls.

<form>
<label for="name">Name:</label>
<input type="text" id="name">
<button type="submit">Submit</button>
</form>
Task 12

Add input types

Implement various input types including email, password, date, checkbox, and radio buttons.

<input type="email">
<input type="password">
<input type="date">
<input type="checkbox">
<input type="radio">
Task 13

Create navigation bars

Build website navigation using lists and the nav element.

<nav>
<ul>
<li><a href="#">Home</a></li>
<li><a href="#">About</a></li>
</ul>
</nav>
Task 14

Use div and span

Apply div and span elements for layout and inline styling purposes.

<div class="container">
Text with <span class="highlight">highlighted</span> part
</div>
Task 15

Use semantic elements

Implement semantic HTML5 elements for better document structure and accessibility.

<header></header>
<main></main>
<section></section>
<article></article>
<aside></aside>
<footer></footer>
Task 16

Create an iframe

Embed external content within your page using iframe elements.

<iframe src="https://example.com" width="500" height="300"></iframe>
Task 17

Create audio and video players

Implement multimedia content with HTML5 audio and video elements.

<audio controls>
<source src="audio.mp3" type="audio/mpeg">
</audio>

<video controls width="320">
<source src="video.mp4" type="video/mp4">
</video>
Task 18

Create anchor links

Build internal page navigation with anchor links and ID references.

<a href="#section1">Go to Section 1</a>

<div id="section1">Section 1 content</div>
Task 19

Add favicons and custom icons

Implement site favicons and integrate custom icon libraries.

<link rel="icon" href="favicon.ico" type="image/x-icon">
Task 20

HTML Entities and Symbols

Use HTML entities to display special characters and symbols.

&lt; displays <
&gt; displays >
&copy; displays ©
&nbsp; for non-breaking space

Advanced

21–30 tasks | ~7–10 days
Task 21

Build multi-page websites

Create a complete website with multiple interconnected HTML pages.

index.html
about.html
contact.html
services.html
Task 22

Apply accessibility

Implement accessibility best practices using alt attributes, ARIA roles, and semantic tags.

<img src="image.jpg" alt="Descriptive text">
<div role="navigation" aria-label="Main menu"></div>
Task 23

Understand rel attribute for SEO

Apply rel attributes for better SEO and security practices.

<a href="https://example.com" rel="nofollow">Link</a>
<a href="https://example.com" rel="noopener noreferrer" target="_blank">Link</a>
Task 24

Use data-* attributes

Implement custom data attributes for JavaScript interaction and metadata.

<div data-user-id="123" data-role="admin">User Content</div>
Task 25

Embed SVG graphics

Integrate scalable vector graphics directly in your HTML using SVG elements.

<svg width="100" height="100">
<circle cx="50" cy="50" r="40" stroke="black" fill="red" />
</svg>
Task 26

Implement responsive HTML structure

Create HTML layouts that adapt to different screen sizes for mobile-friendly design.

<meta name="viewport" content="width=device-width, initial-scale=1.0">
Task 27

Create HTML5 forms with validation

Build forms with built-in validation using HTML5 attributes.

<input type="email" required>
<input type="text" pattern="[A-Za-z]{3,}" title="Three or more letters">
<input type="number" min="1" max="100">
Task 28

Use canvas tag basics

Create dynamic graphics with the HTML5 canvas element and JavaScript.

<canvas id="myCanvas" width="200" height="100"></canvas>

<script>
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.fillRect(10, 10, 100, 50);
</script>
Task 29

Integrate external assets

Connect to external resources like Google Fonts, CDN libraries, and APIs.

<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

<script src="https://cdn.example.com/library.js"></script>
Task 30

Final Project: Build a mini portfolio

Apply all learned concepts to create a complete mini portfolio or blog site using only HTML.

Project Structure:
- index.html (Home)
- about.html
- projects.html
- blog.html
- contact.html
